# 🌾 Agri Inventory Demand Forecasting

A Machine Learning based web application that predicts agricultural inventory demand using historical data and advanced ML models.

---

## 🚀 Features

- 📊 Demand prediction using Machine Learning
- 🌐 Web interface built with Flask
- 🤖 Uses TensorFlow, Scikit-learn, and XGBoost
- 📁 Dataset-driven forecasting
- 🔐 User authentication system (Login/Signup)

---

## 🛠️ Tech Stack

- Python 3.10
- Flask
- TensorFlow
- Scikit-learn
- XGBoost
- Pandas, NumPy
- HTML, CSS (Frontend)

## ⚙️ Installation & Setup

   1. Clone the repository
   git clone https://github.com/madhushankarboya/Agri-inventory-demand-forecasting.git

cd Agri-inventory-demand-forecasting

### 2. Create virtual environment
python -m venv venv
venv\Scripts\activate

### 3. Install dependencies
pip install -r requirements.txt

### 4. Run the application
python run.py

### 5. Open in browser
http://127.0.0.1:10000

## 📈 How it Works

1. User inputs agricultural data
2. Data is processed and sent to ML model
3. Model predicts demand using trained algorithms
4. Result is displayed on the web interface
